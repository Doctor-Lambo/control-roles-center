# Changelog

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), семантика версий — [SemVer](https://semver.org/lang/ru/).

## [2.2.0.1] — 2026-04-19

### Первый публичный релиз

Первая публичная публикация расширения «Центр управления ролями» на GitHub.

### Возможности

- Конструктор пользовательских ролей с детальной настройкой прав доступа
- Поддержка ограничений доступа к данным (RLS) и шаблонов ограничений
- Заполнение прав из ролей конфигурации, профилей групп доступа, пользователей
- Загрузка XML ролей конфигурации в базу
- Ведение прав для нескольких информационных баз
- Автоматическая генерация расширения (CFE) с adopted-объектами
- Автоинкремент версии расширения
- Простой и экспертный режимы редактирования прав
- Работа без БСП с опциональным подключением отдельных методов БСП

### Отличия от оригинала (rarus/custom-role-generator)

- Загрузка XML ролей конфигурации в базу (не только во внешний файл)
- Заполнение прав по пользователю и профилю групп доступа
- Нет ограничения в 99 999 строк на роль (права хранятся в регистре сведений, а не в табличной части)
- Расширенная поддержка RLS — поля и шаблоны ограничений
- Автоинкремент версии расширения
